# CP476 - Product Review and Rating System

## Overview:
A full-stack web application that allows users to add products, submit reviews and ratings, and view aggregated ratings. This project serves as the capstone for CP476 - Internet Computing at Wilfrid Laurier University.

## Features:
- Add products to a database
- Submit reviews and ratings
- View aggregated ratings

## Front End:
- Index is home page for all products and average ratings
- Add products and reviews with "add_review" and "add_product"
- Product lets you view a products details and ratings
- Easily navigated, can access all screens from eachother
- Css and Js folders for styling and code
